# StealthSurf TG Proxy Client ✈️

Легковесный и современный десктопный клиент для автоматической настройки локального Telegram-прокси через шифрованные каналы **StealthSurf**. 

Приложение написано на базе **Tauri v2 (Rust + React)** и использует в качестве фонового ядра высокопроизводительный движок **sing-box**, работающий по протоколу **VLESS Reality** без использования доменного имени (SNI) для максимального обхода блокировок.

---

## ✨ Основные возможности

* **Автоматическое получение конфигураций:** Клиент обращается к официальному API StealthSurf по вашему токену, забирает рабочие конфиги и сам формирует файлы настроек для ядра прокси.
* **Локальный SOCKS5-сервер:** На вашем компьютере (по умолчанию на порту `10808`) поднимается защищенная точка доступа.
* **Интеграция в один клик:** Удобная кнопка автоматически передает ссылку `tg://socks` в официальный клиент Telegram Desktop, настраивая прокси за одну секунду.
* **Работа в фоне (Трей):** Приложение сворачивается в системный трей при закрытии (нажатии на крестик), продолжая работать в фоновом режиме. Левый клик по иконке в трее мгновенно возвращает окно на экран, а правый — открывает удобное меню.
* **Современный UI:** Анимированный неоновый интерфейс в темных тонах с пульсирующим индикатором статуса подключения.

---

## 🛠️ Стек технологий

* **Frontend:** React, TypeScript, HTML/CSS (Inline-стили)
* **Backend:** Rust, Tauri v2, `tauri-plugin-shell`
* **Proxy Core:** sing-box (встроенный как sidecar-бинарник)
